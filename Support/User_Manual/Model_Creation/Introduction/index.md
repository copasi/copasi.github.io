COPASI is a modeling and simulation environment that encompasses the entire
modelling lifecycle. You can create, edit, or import models, simulate them, and
analyze them using a wide variety of methods, which we will cover in this manual.

The COPASI graphical user interface has been developed using the
[Qt application framework](https://www.qt.io/development/qt-framework){:target="\_blank"}.
This allows us to release COPASI on all platforms supported by Qt.

An additional benefit is that COPASI behaves consistently across all supported
platforms, while still accommodating certain platform-specific behaviors. For
example, on computers running macOS, the menu appears at the top of the screen,
and the "About" dialog entry appears in the COPASI menu rather than the Help menu.

In the following sections, we will explain how to use COPASI's graphical user
interface. The information should apply to all supported platforms. If there
are differences specific to certain platforms, we will explicitly point them out.
Should you need additional information, please let us know on our
[Tracker](https://tracker.copasi.org/){:target="\_blank"}.
